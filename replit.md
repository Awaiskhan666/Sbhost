# SelfBot Host

A Discord selfbot hosting platform — a Flask web dashboard that lets users host and manage Discord selfbot accounts.

## Project Structure

- `app.py` — Main Flask web application (dashboard routes, API endpoints)
- `wsgi.py` — WSGI entry point; starts the Flask app and optionally launches main.py bot
- `main.py` — Discord selfbot code (runs as a subprocess per hosted account)
- `models.py` — SQLAlchemy database models (hosted accounts, users, logs, etc.)
- `account_manager.py` — Manages selfbot subprocesses (start/stop/status)
- `config_manager.py` — Centralized configuration with env var overrides
- `crypto.py` — Discord token encryption/decryption
- `enhanced_security.py` — Access control and security utilities
- `templates/` — Jinja2 HTML templates for the dashboard
- `static/` — CSS, JS, and assets

## Deployment (Render)

The app is configured for Render via `render.yaml`:
- Web service runs: `gunicorn wsgi:application`
- PostgreSQL database is auto-provisioned
- `SESSION_SECRET` is auto-generated

### Environment Variables
| Variable | Description |
|---|---|
| `DATABASE_URL` | PostgreSQL connection string (set by Render automatically) |
| `SESSION_SECRET` | Flask session secret (auto-generated by Render) |
| `DISCORD_CLIENT_ID` | Discord OAuth app client ID |
| `DISCORD_CLIENT_SECRET` | Discord OAuth app client secret |
| `DISCORD_REDIRECT_URI` | OAuth redirect URI (e.g. `https://your-app.onrender.com/callback`) |

## Running Locally

```bash
gunicorn wsgi:application
```

Or for development:
```bash
python app.py
```
